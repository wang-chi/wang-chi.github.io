Python 數據分析指令

Read/Write File

Read

import pandas as pd
# read from csv
train = pd.read_csv('application_train.csv')
train = pd.read_excel('application.xlsx')

Write

# export as csv
train.to_csv('train.csv', index=False)

Message Setting

Ignore Warning Message

import warnings
warnings.filterwarnings('ignore')

Show all column data

# 顯示所有資料
import pandas as pd
pd.set_option('display.max_rows', None)
pd.set_option('display.max_column', None)

Data info

  • 檢視特徵型別（非數值型別需要LabelEncoder或者one-hot編碼成數值型）
    app_train.dtypes.value_counts()
  • 統計非數值型資料的unique數
    app_train.select_dtypes('object').apply(pd.Series.nunique, axis = 0)
  • 對齊訓練和測試資料的列數（訓練資料只應該比測試資料多出TARGET一列）
    train_target = app_train['TARGET'] #Target Column
app_train, app_test = app_train.align(other=app_test, join='inner', axis=1)
app_train['TARGET'] = train_target
print('train: ', app_train.shape)
print('test:  ', app_test.shape)

String Method

將換行符號改成空白

a = 'Hello'
b = 3
print(a, end=' ')
print(b)

Result Hello 3

f_string

f_string 使用

a = 'hello'
b = 3
f'{a} {b}'

Result 'hello 3'

f_string 千位數逗號

data = 12500
f'{data:,}'

Result 12,500

f_string控制輸入長度

f'{data:>10,}'

Result ' 12,500'

f_string裡的空白用*填補

f'{data:*>10,}'

Result '****12,500'

f_string的資料對齊方向

f'{data:<10,}'

Result '12,500 '

f_string的資料置中對齊

f'{data:^10,}'

Result ' 12,500 '

將每個字的第一個字元大寫

name = 'peter hsu'
name.title()

Result 'Peter Hsu'

將所有字元大寫

<string>.upper()

將所有字元小寫

<string>.lower()

去除空白

<string>.strip()
左邊
<string>.lstrip()
右邊
<string>.rstrip()

置換字元

replacea to b
<string>.replace(a, b)

變數小技巧

1. 變數  : 純變數，例如：a, b, c
2. 變數(): 小括號表示函數，函數內用逗號區隔，var(a, b, c)
3. 變數[]: 抽取變數內容的元素
4. 變數. ： 使用物件內的屬性或函數

Dict

字典有三種迴圈使用方式items()、keys()、values()

name = {'Billy': 155, 'Chi': 163, 'Vicki': 166}
name.items()

Result
dict_items([('Billy', 155), ('Chi', 163), ('Vicki', 166)])
items()

for k,v in name.items():
    print(k, v)

keys()
使用鍵值k進行迴圈

for k in name.keys():
    print(k, name[k])

values()
使用值進行迴圈，只關心數值

for v in name.values():
    print(v)

Pandas 套件

Pandas 資料結構

  1. Series 單維度欄位
  2. DataFrame 雙維度表格

    Series資料

  • 使用Series()函數建立
  • import pandas
  • 可同時指定index和values

創建Series資料(一)

pd.Series(data=range(1,5), index=list('abcd'))

Result

a    1
b    2
c    3
d    4
dtype: int64

創建Series資料(二)

pd.Series(data=range(1,5))

Result

0    1
1    2
2    3
3    4
dtype: int64

變更索引鍵

rename()

資料的常見用法

  • describe(): 取得資料的描述性統計
  • unique(): 取得資料中不同的資料型態
  • nunique(): n代表number
  • value_counts(): 不同資料型態個別出現次數 (百分比顯示 normalize = True)
  • sort_index(): 按照鍵值大小排序

DataFrame

修改索引列名稱

方法一
需要設定axis=1

df = df.rename({'Math': 'math'}, axis=1)

方法二

df = df.rename(columns={'Math': 'math'})

NaN

檢查資料是否為NaN

import numpy as np
pd.isnull(np.NaN)
pd.notnull(np.NaN)

用dropna()丟棄遺漏值

將遺漏值填滿 fillna()

ffill => fotward fill
bfill => back fill

df.fillna(value=None, mthod=ffill)

列索引鍵取值

  • .loc[index label, column label]: 索引鍵
  • .iloc[index position, column position]:串列用的位置

資料合併

橫向合併

pd.concat([df, df2], axis=1, sort=False)

縱向合併

pd.concat([df, df2], axis=0)
df.append(df2)

使用merge()合併

df.merge(df2, on=’labelname’)

常用函數

印出前後幾筆資料

df.head() #印出前5筆
df.head(3) #印出前3筆
df.iloc[0:3] #印出前3筆
df.tail(3) #印出後3筆
df.iloc[-3:] #印出後3筆
df.sample(3) #隨機3筆資料

資料的資料資料describe()

df.describe()

特定項最高nlargest()

df.nlargest(1,'Math')

df.plot(kind='bar', stacked=True)

rot控制x軸文字角度
水平柱狀圖

df.plot(kind='pie')

df.plot(kind='pie', explode=explode)
